# OGR

Optical Graph Recognition

Trabalho final da disciplina Processamento de Imagens 2015.01 - PESC/COPPE. Aluno: Alexsander Melo. Profs. Antonio Oliveira e Ricardo Marroquim.
Este trabalho consiste na implementação da abordagem proposta em [1] para reconhecimento de desenhos de grafos, cujo o objetivo é extrair suas propriedades topológicas: como vértices, arestas, etc.

[1] Auer, Christopher, et al. "Optical graph recognition." Graph Drawing. Springer Berlin Heidelberg, 2013.
